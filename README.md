# Fact-Check-Agent
A Streamlit-based web application that extracts factual claims from PDF documents and verifies them using Google's Gemini AI.
Features
Upload PDF documents
Extract text from PDFs
Identify factual claims
Verify claims using AI
Generate a verification report
Tech Stack
Python
Streamlit
Gemini API
pdfplumber
Run Locally

pip install -r requirements.txt

streamlit run app.py
